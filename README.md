# Проектная работа: Цифровая трансформация банка "Стандарт"

Проект по архитектуре информационных систем для спринта 3.

## Структура проекта

- **Task1** - Карта IT-ландшафта и схема интеграции приложений
- **Task2** - FURPS+ таблица требований
- **Task3** - ADR для онлайн открытия депозитов (MVP)
- **Task4** - ADR для передачи ставок в кол-центры, задачи и RoadMap
- **Task5** - ADR для онлайн заявок на кредит
